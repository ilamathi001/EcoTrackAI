# EcoTrackAI

AI-Powered Carbon Footprint Awareness Platform

## Overview

EcoTrackAI helps individuals understand, track, and reduce their carbon footprint through AI-powered sustainability insights.

## Features

- Carbon Emission Calculation
- Carbon Score Generation
- AI Sustainability Recommendations
- Activity History Tracking
- PostgreSQL Data Storage
- Gemini AI Integration

## Technology Stack

### Frontend
- Angular

### Backend
- Spring Boot
- Spring Data JPA

### Database
- PostgreSQL

### AI
- Google Gemini AI

### Deployment
- Render
